displayBanner()method
public static function displayBanner(shortenedAppID:String, alignment:String, refreshRate:uint, onSuccess:Function, onFailure:Function, keywords:String = null, age:int = -1, gender:String = null):void

Displays a banner with the given alignment and refresh rate.

Parameters

shortenedAppID:String — The appID to use, without the "_AutoDeviceType" part added by Inneractive
 
alignment:String — One of the ALIGN_constants
 
refreshRate:uint — The number of seconds between ad refreshes (min:15, max:300)
 
onSuccess:Function — Called when the banner is displayed. Signature : function(isPaidAd:Boolean):void
 
onFailure:Function — Called when displaying the banner fails for any reason. Signature : function():void
 
keywords:String (default = null) — Relevant keywords for ad targeting. For example: cars,music,sports (comma separated, w/o spaces)
 
age:int (default = -1) — The user's age (between 1 and 120). Leave to -1 to ignore this parameter
 
gender:String (default = null) — The user's gender. One of the GENDER_constants

Fetches an interstitial ad with the given parameters.

Parameters

shortenedAppID:String — The appID to fetch an interstitial for, without the "_AutoDeviceType" part added by Inneractive
 
onFetched:Function — Called when the ad is successfully fetched. Signature : function(isPaidAd:Boolean):void
 
onFailure:Function — Called when the ad fails to be fetched. Signature : function(errorMessage:String):void
 
keywords:String (default = null) — Relevant keywords for ad targeting. For example: cars,music,sports (comma separated, w/o spaces)
 
age:int (default = -1) — The user's age (between 1 and 120). Leave to -1 to ignore this parameter
 
gender:String (default = null) — The user's gender. One of the GENDER_constants
